## Overview

TerosHDL is an open-source tool that can help you create, simulate, and synthesize your designs. It is built on top of the open-source tools such as GHDL, GTKWave, and Yosys.

## Installation

TerosHDL can only be used with VSCodium or VSCode.

## Usage

Once installed, you can start using TerosHDL by opening the application. The application provides a graphical user interface that allows you to create, simulate, and synthesize your designs.
